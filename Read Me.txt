Install Localhost:

1. Buat database dengan nama db_toko di phpmyadmin.
2. Masukan isi folder kedalam directori:
   C:\\wamp\www (jika menggunakan WAMPSERVER).
   C:\\xampp\htdocs (jika menggunakan XAMPP).
   NB: PAstikan anda mengaktifkan Rewrite Module pada webserver Anda.!
3. Buka http://localhost/phpmyadmin. Lalu import db_toko.sql kedalam database yang anda buat.
   NB: file db_toko.sql terdapat pada folder duamasa
4. Pada Browser Buka URL http://localhost/duamasa
5. Selesai.
